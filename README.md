# Music_downloader
Music_downloader by Python Spider.

Python课程大作业。

使用爬虫进行歌曲下载，包括酷狗音乐、酷我音乐、QQ音乐、网易云音乐和咪咕音乐。


| 进度 | 酷狗音乐 | 酷我音乐 | QQ音乐 | 网易云音乐 | 咪咕音乐 |
| :----: | :----: | :----: | :----: | :----: | :----: |
| 歌曲搜索 | √ | √ | √ | - | - |
| 歌曲下载 | √ | √ | - | - | - |

运行
```python
python3 main.py
```
